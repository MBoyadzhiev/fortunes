# Настройка на Баркода за Магазина

## Как да настроиш баркода

### Стъпка 1: Избери баркод стойност

Отвори файла `src/config.js` и промени стойността на `SHOP_BARCODE`:

```javascript
export const SHOP_BARCODE = 'ТВОЯТ_БАРКОД_ТУК'
```

Например:
```javascript
export const SHOP_BARCODE = 'KUSMETI2024'
```

### Стъпка 2: Създай баркод/QR код

Използвай онлайн генератор на QR код (например https://www.qr-code-generator.com/) или баркод генератор и създай QR код/баркод с избраната стойност.

### Стъпка 3: Разпечатай и постави в магазина

Разпечатай баркода и го постави на видно място в магазина, където клиентите могат лесно да го сканират.

### Стъпка 4: Тествай

1. Стартирай приложението: `npm run dev`
2. Отвори на мобилно устройство
3. Сканирай баркода
4. Ако баркодът съвпада, ще се покаже съдба
5. Ако баркодът не съвпада, ще се покаже съобщение за грешка

## За Vercel Deployment

Ако искаш да използваш environment variable вместо да променяш кода:

1. Отиди в Vercel Dashboard → Settings → Environment Variables
2. Добави нова променлива:
   - Name: `VITE_SHOP_BARCODE`
   - Value: твоята баркод стойност
3. Redeploy приложението

Това е по-безопасно, защото баркодът няма да е видим в кода.

## Важни бележки

- Баркодът се сравнява точно (case-sensitive)
- Празните пространства в началото и края се премахват автоматично
- Всеки клиент трябва да сканира същия баркод, за да получи съдба
- След успешно сканиране, всеки получава случайна съдба от колекцията
